Program

7 months to deployment-ready.

A modular program that takes a complete beginner to an industry-ready cybersecurity professional. Available as standalone modules or as a complete career track.

Phase 1 · Months 1–3

Foundation

Mandatory for all students. Builds the technical base every cybersecurity professional needs: OS internals, networking, traffic analysis, security frameworks, and Python scripting.

  • Windows & Linux system administration from scratch
  • Build and manage your own virtual cybersecurity lab
  • Analyze real network traffic with Wireshark
  • OWASP Top 10, NIST, and ISO 27001 frameworks
  • Python automation scripts for security tasks
  • 48-hour Foundation Security Audit Capstone

Phase 2 · Months 4–6

Specialization

Select one of four tracks. Three months of advanced, job-specific training in the chosen domain.

Phase 3 · Month 7

Career Launch

Not coursework. Career deployment.

  • 72-hour live capstone across all four tracks simultaneously
  • Portfolio finalization: GitHub, reports, dashboards
  • ATS resume optimization and LinkedIn transformation
  • Mock interviews: HR screen, technical panel, whiteboard, stress-test
  • Partner Company Demo Day with hiring managers
  • Alumni network for ongoing referrals and certification support

Specialization Tracks

01

SOC Analyst · 12 Weeks

Security Operations

Detect, triage, and respond to threats using enterprise SIEM tools and threat intelligence. Splunk, Microsoft Sentinel, MITRE ATT&CK, Zeek, Suricata, and NIST 800-61 incident response.

Deliverables: SOC Dashboard, Alert Triage Lab, IR Report, Threat Hunt Report

02

VAPT · 12 Weeks

Penetration Testing

Offensive security and ethical hacking. OSINT with Maltego, Nmap, Nessus, Metasploit, Burp Suite Pro, Active Directory attacks with BloodHound and Mimikatz.

Deliverables: OSINT Profile, Web App Pentest Report, 15-Page VAPT Report

03

Cloud Security · 12 Weeks

Cloud Infrastructure

Secure cloud infrastructure on AWS and Azure. IAM policies, Zero Trust design, container/Kubernetes hardening, GuardDuty, Checkov, Prowler compliance auditing.

Deliverables: IAM Remediations, Hardened Container Lab, Compliance Audit Report

04

DFIR · 12 Weeks

Digital Forensics

Forensic investigation and incident response. Chain of custody, FTK Imager, NTFS forensics, Volatility 3 memory analysis, PEStudio malware analysis, and full breach investigation.

Deliverables: Forensic Image, Attack Timeline, 20-Page Breach Investigation Report

FAQ

Common questions.

Who can enroll?

Any college student, recent graduate, or IT professional. No prior cybersecurity experience required.

Is it live or recorded?

Training is live and interactive. Every session involves instructor delivery, Q&A, and real-time lab work.

Can I take only Foundation?

Yes. All phases are available as standalone modules. No need to commit to the full 7-month track.

Do you guarantee placement?

We provide structured placement support: resume building, mock interviews, demo days, and direct company referrals.

Do I need a laptop?

Yes. At least 8GB RAM and 100GB free disk space for virtual lab environments. Setup guidance provided from Day 1.

How do I enroll?

Contact the Zharnyx team directly. Enrollment slots are limited per cohort to ensure quality mentorship.
